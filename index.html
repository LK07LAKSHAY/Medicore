Data Resilience

Backup & Restore

Export all clinic data to a single JSON file. Restore it on any device — no server needed.

💾 Local Data Snapshot
All data — patients, referrals, packages, and bookings — lives in your browser's localStorage. Export a backup regularly and import it to restore or migrate to another machine.

Core Feature
🔍
Duplicate Detection
Automatically flags duplicate patient entries by mobile number before saving, preventing data redundancy.
💾
Persistent Storage
All patient data and referrals are persisted in localStorage — no database setup needed, works offline.
Core Feature
📦
Test Packages
Bundle multiple tests into packages with custom pricing, showing savings over individual test prices.
💰
Referral Margins
Configure per-doctor referral margins as fixed amount or percentage, auto-applied during billing.
🧫
Specimen Management
Assign specimen types per-test or globally across all patient tests with a multi-select picker.
📱
Responsive Design
All four modules are fully responsive and work seamlessly on desktop, tablet, and mobile devices.
